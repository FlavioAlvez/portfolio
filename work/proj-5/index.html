Retrieved data via the Dribbble API formatted into HTML. Dribbb includes a full-width list view and a project view. The JavaScript was written in ES6 and JSX, transpiled with Babel, and polyfilled. The JavaScript adheres to MVC conventions with Backbone serving as Model and Controller, and React handling View. Source code available on GitHub.

Built With: HTML5, Sass, JavaScript, Backbone, React
